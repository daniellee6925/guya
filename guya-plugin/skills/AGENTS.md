# skills

Skills for personal evolution, memory management, decision-making harnesses, code quality, and workflow.

## Purpose

Skills are reusable workflow templates that can be invoked via `/guya-skill-name`. Each skill provides:
- Interactive workflows for identity building, memory management, and decision-making
- Integration with Guya's evolution system
- Session-scoped and cross-session learning
